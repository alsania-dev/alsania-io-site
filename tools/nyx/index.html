What is Nyx?

Nyx is a powerful browser extension that integrates Model Context Protocol (MCP) tools directly with your favorite AI platforms.

Execute MCP tools from within AI conversations, enhancing both free and paid web-based AI assistants with real-time automation capabilities.

Now available for Chrome, Edge, and Firefox!
